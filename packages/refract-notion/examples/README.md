# Demo CSS

이 폴더의 CSS는 **참고용**입니다.  
라이브러리 npm 패키지에는 포함되지 않습니다.

## 사용 방법

1. `demo.css` 내용을 프로젝트에 복사
2. 필요에 맞게 수정

## CSS 클래스 목록

### 공통
- `.notion-block` - 모든 블록 공통
- `.notion-indent` - 하위 블록 들여쓰기 (children wrapper)

### Typography
- `.notion-h1`, `.notion-h2`, `.notion-h3` - 헤딩
- `.notion-paragraph` - 문단
- `.notion-quote` - 인용
- `.notion-list`, `.notion-ul`, `.notion-ol` - 리스트
- `.notion-list-item` - 리스트 아이템

### Media & Decoration
- `.notion-image` - 이미지
- `.notion-callout` - 콜아웃
- `.notion-bookmark` - 북마크
- `.notion-divider` - 구분선

### Advanced
- `.notion-toggle` - 토글
- `.notion-code-block` - 코드 블록
- `.notion-table` - 테이블

### Colors
- `.notion-color-{color}` - 텍스트 색상 (e.g., `notion-color-orange`)
- `.notion-bg-{color}` - 배경 색상 (e.g., `notion-bg-blue`)
